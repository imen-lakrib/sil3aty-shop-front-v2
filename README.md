# Updated sil3aty-shop V-2

Welcome to the updated version of the **sil3aty-shop** repository! This repository features a new design, powerful features, and a TypeScript implementation, bringing an improved user experience, advanced functionality, and enhanced code quality to the project.

## Overview

This repository contains the frontend codebase for the **sil3aty-shop** project, which is hosted at [sil3aty-shop-front-v2.vercel.app](https://sil3aty-shop-front-v2.vercel.app). The project aims to provide a modern and engaging online shopping experience to users.

## Features

- **New Design**: The frontend has been revamped with a fresh and modern design, enhancing the visual appeal and usability of the website.

- **Powerful Features**: The project includes a range of powerful features, such as a user-friendly product catalog, secure shopping cart, efficient checkout process, and more.

- **TypeScript Implementation**: The codebase has been migrated to TypeScript, providing better code organization, type safety, and improved development experience.

## Getting Started

To run the project locally, follow these steps:

1. Clone this repository to your local machine.
   ```bash
   git clone https://github.com/imen-lakrib/sil3aty-shop-front-v2
   ```

2. Navigate to the project directory.
   ```bash
   cd sil3aty-shop
   ```

3. Install the project dependencies.
   ```bash
   npm install
   ```

4. Start the development server.
   ```bash
   npm run dev
   ```

5. Open your browser and visit [http://localhost:5173/](http://localhost:5173/) to access the locally running application.
